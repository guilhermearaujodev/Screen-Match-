# Screen-Match-part-3

O projeto simula uma plataforma de streaming.

A part-3 do projeto tem como foco consumir uma API HTTP em Java, receber e converter informações de uma API para objetos Java, manipular dados em formato JSON, conhecer a biblioteca Gson e usar classes do pacote java.io.

Esse projeto faz parte do curso da Alura "Java: consumindo API, gravando arquivos e lidando com erros"
